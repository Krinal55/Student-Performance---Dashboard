# 🎓 Student Performance Dashboard

 📌 Overview
   This project presents an interactive Student Performance Dashboard built using Microsoft Power BI.
   The dashboard provides insights into student academic performance, attendance, and subject-wise analysis to help educators make data driven decisions.

---

## 🚀 Project Overview
- 📊 Student academic performance    
- 🧠 Behavior patterns  
- 🏫 Class & section-wise insights  

It transforms raw student data into meaningful visual insights for better decision-making.

---

## 🧩 Features 

- 📈 Performance Metrics:
  - Total Students  
  - Attendance %  
  - Average Score  

- 📊 Visual Insights:
  - Score by Term  
  - AVG Score by Subject  
  - Behavior Count Distribution  

- 📋 Detailed Tables:
  - Student Scores  
  - Student Behavior Details  

- 🧠 Behavior Analysis:
  - Disruptive  
  - Participative  
  - Helpful  
  - Late  
  - Absent without notice
