# 🎓 Student Performance Dashboard

An interactive **Power BI dashboard** designed to analyze student performance, attendance, and behavior insights in a clear and actionable way.

---

## 🚀 Project Overview

This dashboard helps educators and administrators monitor:

- 📊 Student academic performance  
- 📅 Attendance trends  
- 🧠 Behavior patterns  
- 🏫 Class & section-wise insights  

It transforms raw student data into meaningful visual insights for better decision-making.

---

## 🧩 Features

- 🎯 Dynamic Filters:
  - Class  
  - Section  
  - Subject  
  - Term  

- 📈 Performance Metrics:
  - Total Students  
  - Attendance %  
  - Average Score  

- 📊 Visual Insights:
  - Score by Term  
  - AVG Score by Subject  
  - Behavior Count Distribution  

- 📋 Detailed Tables:
  - Student Scores  
  - Student Behavior Details  

- 🧠 Behavior Analysis:
  - Disruptive  
  - Participative  
  - Helpful  
  - Late  
  - Absent without notice
